## Estrutura de Nomenclatura

Organize as imagens usando o slug do artigo seguido do tipo:
- `[slug]-featured.jpg` - Imagem de destaque (obrigatória)
- `[slug]-[descricao].jpg` - Imagens de conteúdo (opcionais)

**Exemplos:**
- `curva-crescimento-infantil-featured.jpg`
- `curva-crescimento-infantil-grafico.jpg`
- `diabetes-tipo1-escola-featured.jpg`
- `diabetes-tipo1-escola-lancheira.jpg`

## Especificações Técnicas

### Featured Images (Imagens de Destaque)
- **Tamanho**: 1200x630px (proporção Open Graph)
- **Formato**: JPG ou WebP (otimizado)
- **Peso**: Máximo 200KB (recomendado < 150KB)
- **Uso**: Meta tags Open Graph, Twitter Cards, listagem do blog

### Imagens de Conteúdo
- **Largura**: 800-1200px (conforme necessidade)
- **Formato**: JPG, PNG ou WebP
- **Peso**: Máximo 300KB (recomendado < 200KB)
- **Uso**: Dentro do conteúdo do artigo

### Infográficos
- **Tamanho**: Conforme design (geralmente 1200px de largura)
- **Formato**: PNG (com transparência) ou SVG
- **Uso**: Ilustrações e gráficos explicativos

## Otimização

**Sempre otimize as imagens antes de adicionar:**
- Use ferramentas como TinyPNG, ImageOptim ou Squoosh
- Converta para WebP quando possível (melhor compressão)
- Redimensione para o tamanho exato necessário
- Mantenha qualidade alta mas arquivo pequeno

## Alt-Text

Cada imagem deve ter alt-text descritivo em português quando utilizada no código.

**Exemplos de alt-text:**
- `alt="Gráfico de curva de crescimento infantil mostrando percentis de altura"`
- `alt="Criança sendo medida por pediatra para acompanhamento de crescimento"`
- `alt="Lancheira saudável com alimentos balanceados para criança com diabetes"`

## Como Usar no Código

Em Next.js, use o caminho relativo a `/images/blog/`:

```tsx
// Imagem de destaque
<img src="/images/blog/curva-crescimento-infantil-featured.jpg" 
     alt="Curva de crescimento infantil" />

// Ou usando o componente Image do Next.js (recomendado)
import Image from 'next/image';

<Image 
  src="/images/blog/curva-crescimento-infantil-featured.jpg"
  alt="Curva de crescimento infantil"
  width={1200}
  height={630}
/>
```
