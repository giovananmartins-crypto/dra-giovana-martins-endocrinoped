# Pasta de Imagens do Site

Esta pasta contém todas as imagens utilizadas no site da Dra. Giovana Martins.

## Estrutura de Pastas

### `/dra-giovana/`
Imagens da Dra. Giovana Martins:
- Foto profissional para homepage
- Foto para seção "Sobre"
- Fotos para redes sociais (Open Graph, Twitter Cards)
- Exemplo: `dra-giovana-foto-profissional.jpg`

### `/blog/`
Imagens dos artigos do blog:
- Imagens de destaque (featured images)
- Imagens de conteúdo dos artigos
- Infográficos
- Exemplo: `blog/curva-crescimento-infantil.jpg`

### `/servicos/`
Imagens das páginas de serviços:
- Imagens de destaque para cada serviço
- Ilustrações médicas
- Exemplo: `servicos/puberdade-precoce.jpg`

### `/home/`
Imagens da homepage:
- Imagens de seções específicas
- Banners
- Exemplo: `home/hero-background.jpg`

### `/icons/`
Ícones e logos:
- Favicon
- Logos
- Ícones de redes sociais
- Exemplo: `icons/favicon.ico`

## Convenções de Nomenclatura

- Use nomes descritivos em português
- Use hífens para separar palavras (kebab-case)
- Exemplo: `dra-giovana-foto-profissional.jpg`
- Exemplo: `blog/curva-crescimento-infantil-featured.jpg`

## Formatos Recomendados

- **Fotos**: JPG ou WebP (otimizado)
- **Ilustrações/Infográficos**: PNG ou SVG
- **Ícones**: SVG ou PNG
- **Tamanho recomendado para featured images**: 1200x630px (Open Graph)
- **Tamanho recomendado para imagens de conteúdo**: 800-1200px de largura

## Otimização

- Sempre otimize as imagens antes de adicionar
- Use ferramentas como TinyPNG, ImageOptim ou similar
- Mantenha arquivos pequenos para melhor performance
- Use WebP quando possível para melhor compressão

## Alt-Text

Todas as imagens devem ter alt-text descritivo em português quando utilizadas no código.
