# Ícones e Logos

Esta pasta contém ícones e logos do site.

## Ícones Necessários

- `favicon.ico` - Favicon do site
- `logo.svg` ou `logo.png` - Logo da Dra. Giovana Martins (se houver)
- Ícones de redes sociais (se necessário)

## Especificações

- **Favicon**: 32x32px ou 16x16px, formato ICO
- **Logo**: SVG (preferencial) ou PNG com fundo transparente
- **Tamanho**: Conforme necessidade
